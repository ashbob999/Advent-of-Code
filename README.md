# Advent-of-Code
All solutions to the Advent of Code days. (That I've completed)


to generate day files:
```shell script
python generate.py
python aoc/generate.py
```

options:
```shell script
-y (--year)
-d (--day)
```

to run day file:
```shell script
python -m aoc
```

options:
```shell script
-y (--year)
-d (--day)
-s (--session)
-i (--input)
```
